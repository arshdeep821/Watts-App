# Watts-App

### NW-Hacks 2024 Project

## Front-End

```zsh
cd client
yarn install
yarn dev run
```

## Back-End

- Running Server
```zsh
cd server
npm install
npm start
```

- Running Script
```zsh
cd server
cd scripts
node script.js
```
